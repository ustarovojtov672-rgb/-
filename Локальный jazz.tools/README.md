# Локальный jazz.tools

Эта папка фиксирует локальный режим Jazz для приложения.

- Данные хранятся в IndexedDB браузера.
- Синхронизация с Jazz Cloud отключена через `sync.when = "never"`.
- Схема дневника питания лежит в `../src/lib/jazz/schema.ts`.
- Provider подключен в `../src/components/local-jazz-provider.tsx`.

Если позже понадобится облачная синхронизация, лучше добавить отдельную миграцию конфигурации, а не смешивать локальный и облачный режимы.
